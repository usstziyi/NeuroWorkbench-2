# NeuroWorkbench

EEG 采集、处理、显示、AI 计算一体化桌面客户端。

## 技术栈

- **GUI 框架**: PySide6
- **绘图库**: pyqtgraph
- **语言**: Python 3

## 目录结构

```
NeuroWorkbench-2/
├── main.py          # 主入口，主窗口及 UI 布局
└── README.md
```

## 安装依赖

```bash
pip install PySide6 pyqtgraph
```

## 运行

```bash
python main.py
```

## 左侧控制面板

通过 `pyqtgraph.parametertree.ParameterTree` 提供三类参数控制：

- **设备设置**: 采样率、通道数、设备类型
- **滤波设置**: 陷波滤波器开关、陷波频率、高低通截止频率
- **显示设置**: 时间窗口、Y 轴范围

三个控制按钮：开始采集、停止采集、开始记录。
